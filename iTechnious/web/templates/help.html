Hilfeseite zur Hackathon Verwaltung

Die Startseite

Auf der Startseite findest du oben den Menüband. Der Knopf "Server anmelden" öffnet den Discord Dialog, um den Bot zu einem Eventserver hinzuzufügen. Daneben findest du ein Dropdown mit den Events, die aktuell laufen.
Zu der Eventübersicht findest du weiter unten noch etwas.

Einen Eventserver anmelden

Bei einem Klick auf den "Server anmelden" Knopf öffnet sich ein neuer Tab und du wirst auf die Discord Seite weitergeleitet. Dort kannst du einen Server auswählen, auf den der Bot eingeladen werden soll. Nachdem du einen ausgewählt hast, musst du dem Bot noch Admin Rechte bestätigen und ein CAPTCHA lösen. Danach passieren einige Dinge. Nach einem Klick auf "Authorize" kommt der Bot auf den Server. Dort erstellt er einen Event-Manager Kanal und die zugehörige Rolle. Diese Rolle "Event-Manager" kannst du dir selbst und deinen Mit-Verwaltern geben. Dadurch bekommen sie den Adminzugang auf dieser App.

Eventserver Übersicht

Auf der Server Übersicht findest du als erstes den Namen und das Icon des Wettbewerbs. Außerdem findest du den Einladungslink zum entsprechenden Discord Server.
Solltest du noch keinem Team beigetreten sein, kannst du mit dem Knopf oben links eins erstellen. Solltest du zusätzlich die Event-Manager Rolle haben, kannst du ebenso auf den Admin Modus wechseln. Zu dieser Seite aber gleich mehr.
Unter diesem ersten Abschnitt findest du die Übersicht der Teams. Mit einem Knopfdruck auf "Team beitreten" beim entsprechenden Team kannst du eine Anfrage zum Team schicken. Diese kann dann von jedem der Teammitglieder, oder von den Event-Managern angenommen oder abgelehnt werden. Sobald du einem Team angehörst, wird dieses Team dir ganz oben angezeigt. Solltest du es wieder verlassen wollen, so ist dies auch mit einem Knopfdruck auf "Team verlassen" möglich. Damit wir alle über diese Vorgänge auf dem Laufenden halten können, solltest du schon vor dem Anmelden auf den Server joinen, da der Bot nur Nachrichten an Nutzer senden kann, die auch auf dem Server sind.

Beitritts-Anfragen

Sollte dein eigenes Team Beitritts-Anfragen erhalten, dann werden sie ebenfalls auf der Übersicht erscheinen. Es werden die entsprechenden Discord Namen mit den Knöpfen für annehmen und ablehnen angezeigt. Klickst du auf "Annehmen", so wird dieser Benutzer aus der Schwebe genommen und deinem Team hinzugefügt. Dabei bekommt der Benutzer gleichzeitig die Team-Rolle auf dem Discord Server. Sollte das Mitglied erst danach auf den Server joinen ist das kein Problem. Dem Benuter wird die Team-Rolle automatisch hinzugefügt.

Admin Modus

Jedes Server Mitglied mit der "Event-Manager" Rolle, hat die Möglichkeit in den Admin Modus zu wechseln. In dieser Ansicht werden noch unbestätigte Teams ganz oben ausgeführt. Bei diesen Teams besteht dann die Möglichkeit der Freischaltung oder des Löschen. Bei letzterer Variante wird der Eintrag gelöscht und der Ersteller benachrichtigt. Beim Bestätigen werden die Teamkanäle und eine Teamrolle erstellt. Die Kanäle sind nur für die Rollenbesitzer einsehbar. Sollte der Ersteller des Teams bereits auf dem Server sein, wird ihm die Rolle auch zugewiesen. Sollte ein Mitglied noch nicht auf dem Server sein, aber bereits einem Team angehören, wird beim joinen die entsprechende Rolle automatisch zugewiesen.
Die zweite Option im Admin Modus ist, Beitritts-Anfragen für jedes Team anzunehmen und abzulehnen. Dies hat den gleichen Effekt, wie die Bestätigung als Teammitglied.
Die dritte Option im Admin Modus ist das löschen von Teams. Diese Funktion ist einzig und allein den Administratoren vorbehalten. Beim Löschen wird das Team aus dem Webinterface entfernt und die Teamrolle gelöscht. Allerdings bleiben die Teamchannel bestehen und werden mit einem roten Kreuz versehen, so kann Datenverlust vergebeugt werden. Die Kanäle müssen manuell über Discord gelöscht werden.

Immer noch Fragen?

Dann nimm gerne Kontakt mit uns auf! Wir werden so schnell wie möglich auf gemeldete Fehler, Feedback, allgemeine Fragen und alles Andere antworten! Verwende einfach unser Kontaktformular.

F.A.Q

Deine Fragen kurz und bündig beantwortet!

  • Muss ich mir ein Konto erstellen?

    Hier musst du dir kein Konto erstellen. Zweckmäßig ruft diese App aber deine Kontodaten von Discord ab. Dies passiert, sobald du auf die Events zugreifen möchtest. Dazu musst du dich bei Discord anmelden bzw. registrieren.

  • Kann ich in mehreren Teams gleichzeitig sein?

    Nein, das ist offiziell nicht möglich.

  • Was, wenn ich in einem Team bin, aber noch nicht auf dem Server bin?

    Absolut kein Problem! Klicke einfach auf den Link unter dem Namen des Events und joine so dem Eventserver. Wenn du joinst, wird dir automatisch deine Team-Rolle zugewiesen.
